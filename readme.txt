The "Booking Commerce Plugin" adds a booking widget to WordPress, letting admins choose display pages for seamless booking portal integration.

== Description ==
The "Booking Commerce Plugin" is a comprehensive solution that adds a booking widget to your WordPress site, allowing users to interact with a booking commerce portal seamlessly. This plugin enables administrators to select specific pages to display the booking widget, providing an integrated and streamlined booking experience for visitors. Ideal for businesses that require a booking interface on their site, this plugin enhances functionality by connecting your WordPress site directly with your booking portal.

== Short Description ==
A powerful WordPress plugin that embeds a booking widget for your commerce portal, allowing page-specific widget placement and easy integration with your site.

== Features ==

1. **Booking Widget Integration:** Adds a fully functional booking widget for commerce portals, making it easy for users to book services or products directly on your website.
2. **Page Selection Control:** Allows administrators to choose specific pages where the booking widget will be displayed, offering flexibility in placement.
3. **Easy Configuration:** Straightforward setup and configuration process to help you get the widget running in no time.
4. **Seamless User Experience:** Provides an embedded booking experience, keeping users on-site while accessing your booking portal.
5. **Customizable and Lightweight:** Designed to be efficient and lightweight, minimizing impact on site performance while providing essential booking functionality.

== Installation ==
1. Upload the `booking-commerce-plugin` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the plugin settings to select the page(s) where you want the booking widget to appear.

== Frequently Asked Questions ==

= Can I place the widget on multiple pages? =
Currently, the widget can be displayed on one selected page. Future updates may support multiple-page selection.

= Does the plugin support custom styling for the widget? =
The widget inherits styling from your theme, but you can apply custom CSS in your theme settings or via custom code.

= Is this plugin compatible with all WordPress themes? =
Yes, the plugin is designed to work with most themes, though some custom themes may require additional CSS adjustments.
